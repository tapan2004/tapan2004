## 🚀 Featured Projects

Here are a few high-impact projects I've developed, showcasing enterprise architecture and modern full-stack practices:

### ☕ [Smart Cafe Management System](https://github.com/tapan2004/smart-cafe-management-system)
*AI-Powered Cafe Operations and Sales Intelligence Engine*
- **Stack**: Java, Spring Boot, FastAPI, MySQL, Python.
- **Key Features**: Implemented intelligent sales forecasting, a real-time product recommendation system, and automated inventory management workflows.
- **Why it matters**: Demonstrates cross-language service communication (Java $\leftrightarrow$ Python/FastAPI) and machine learning integration into a traditional business system.

### ⚡ [Asynchronous Order Processing System](https://github.com/tapan2004/spring-boot-async-order-processing)
*High-Throughput Non-Blocking Transaction Pipeline*
- **Stack**: Java, Spring Boot, `@Async`, ThreadPoolTaskExecutor, CompletableFuture.
- **Key Features**: Designed custom thread-pool executors to process massive order transactions concurrently, avoiding thread exhaustion and keeping request flows non-blocking.
- **Why it matters**: Showcases deep understanding of JVM concurrency, asynchronous programming, thread-pool configuration, and fallback/error handling under heavy load.

### 🛡️ [Enterprise Auth & Authz Service](https://github.com/tapan2004/spring-boot-security-authentication-authorization)
*Production-Ready RBAC and JWT Authentication System*
- **Stack**: Spring Boot, Spring Security, JWT, PostgreSQL.
- **Key Features**: Standardized Role-Based Access Control (RBAC), secure token generation, expiration, validation lifecycle, and BCrypt encrypted credentials.
- **Why it matters**: Highlights security-first development practices, robust middleware filters, and secure database credential lookup.

### 📊 [Smart Finance Tracker API](https://github.com/tapan2004/SmartFinanceTrackerApi)
*Full-Stack Financial Management Platform*
- **Stack**: React, Spring Boot, MySQL, JavaMailSender, JWT.
- **Key Features**: Income/expense tracking system featuring secure token authorization and background threads for dispatching automated email analytics reports.
- **Why it matters**: Exhibits end-to-end full-stack synchronization, state management, and async notifications.
